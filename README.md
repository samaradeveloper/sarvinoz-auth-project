## Project description
Small authentication demo: Login page, client validation, DB connection, user registration, and forgot-password UI.

## How to run (local)
1. Clone: `git clone https://github.com/<owner>/<repo>.git`
2. Backend: `cd backend` -> `npm install` -> `node app.js`
3. Frontend: open `frontend/login.html` in browser or use simple static server.
